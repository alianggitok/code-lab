[ 文字超出省略的处理脚本应用 ]

用法说明

uaxpUI.wordsEllipsis(obj,ellipsisSymb,adjustLen);

原理

逐字递减，判断文字高度是否超出父元素高度，如果超出则继续减少文字，直至不再超出。基于jQuery。

参数

obj: 应用对象
ellipsisSymb: 省略符号html
adjustLen: 修正省略字长

必须

需在html文档中引用“jquery-ua-common.js”

需定义对象style的height或max-height值（一般为line-height×行数），且设置overflow为hidden

示例

CSS

.words{
	max-height:48px;
	line-height:16px;
	font-size:12px;
	overflow:hidden
}
.ua-ellipsissymb{color:red}

Javascript

$(function(){
	uaxpUI.wordsEllipsis('.words','<span class="ellipsissymb">…<span>',10);
});

HTML

<p class="words">
	文字……文字
</p>
